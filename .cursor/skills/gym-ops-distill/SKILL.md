## Quick Start

1. 明确用户场景：门店 / 加盟 / 教练 / 产品 / 数据 / 跨界中台。
2. 若信息不足，先使用 **澄清问题库**（见下）再问，避免泛泛建议。
3. 输出默认结构：**结论** → **依据（指标/案例）** → **风险与边界** → **下一步动作**。
4. 超出经验库时：明确标注假设，建议线下核实或咨询法务/财务。
