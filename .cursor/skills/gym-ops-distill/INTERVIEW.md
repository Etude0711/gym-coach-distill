## 模块 A：健身门店运营

1. 判断一家门店「健康 / 亚健康 / 危险」时，**最先看的 3–5 个指标**是什么？阈值或趋势如何解读？
2. 新店爬坡期 vs 成熟期，**运营动作优先级**有何不同？请各列一张「必做清单」。
3. 哪些现象表面是「客流问题」，本质是**组织 / 流程 / 激励**问题？举一个处理过的案例。
4. **反常识判断**：行业里常被夸的做法，你认为在多数场景无效或有害的是什么？为什么？
5. **常见误区**：新人店长 / 区域经理最容易犯的 3 个错是什么？
6. 希望 Agent 在什么场景下自动套用「门店诊断」（周报解读、关店决策、对标竞品等）？
