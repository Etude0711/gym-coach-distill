## PRIORITY（已完成）

### P1. 技能定位与受众

- **主要服务**：健身行业**同行**，以及对行业**感兴趣**的人（投资人、从业者视角均可）。

### P2. 业态与经验边界

- **适用业态**：直营、连锁加盟的**综合健身房**、**私教工作室**、**团课工作室**。

**不写 / 强免责 / 转介**（详见 `references/pitfalls/scope-boundaries.md`）：

| 场景 | 处理 |
|------|------|
| 医疗康复 | 找康复科 / 运动医学，skill 不替诊 |
| 少儿体适能 | 严格资质，不给具体方案 |
| 海外市场 | 可交流原则；国内做的人少，器械出口为主，勿外推国内模型 |
| 瑜伽 / 普拉提细分 | 非核心战场，仅泛化聊 |
| 纯线上课 | **不看好** 作主模式 |
| 器械销售等 | 微信 **anachronism** |

### P3. 触发场景（框架/入口）

**总入口（用户原话）**：自查健身房的经营，先问问这些问题。

| 框架名 | 覆盖问题 | 详述 |
|--------|----------|------|
| **回报率** | 投资回报、建店造价 | `references/playbooks/self-check.md` §1 |
| **门店运营数据，会员卡和私教** | 三表 + 在期/拉新/消课 | §2 |
| **教练管理要素** | 激励、人均消课 | §3 |
| **CRM，用户管理** | 流失分层、召回 | §4 |
| **门店产品设计与体验设计** | 课包、合规、体验 | §5 |

建议自查顺序：回报率 → 门店运营数据 → CRM → 教练管理要素 → 产品设计体验。

### P4. 门店经营判断

按**财务报表逻辑**看经营，而非单点 KPI：

1. **资产负债表** — 结构是否健康
2. **损益表** — 赚不赚钱、成本结构
3. **现金流量表** — 能不能活下去

- **损益表（已答）**：`references/metrics/store-financials.md` §一、二
- **资产负债表（已答）**：负债看月度实消能否盖固定成本；应付款重能耗；器械折旧 **24–30 月** 较合理（§三）
- **现金流/合规/季节（部分已答）**：上海预收监管、3–8 月旺季、淡季现金流、季度大促（§四）
- **现金流盯盘（已答）**：在期≥450、拉新 2–3 人/日、流失分层 50%/30%、私教人均消课 <100 节/月查管理、新店预售 20–50 万、商圈 vs 社区渗透周期（§四 实操盯盘）

### P5. 加盟（选址 → 招商）

**选址（已答）** → `references/judgments/franchise.md`

- 优先商场、商业综合体；2025 年后房租普遍降 40–50%
- 贝壳估人口 + 夜访（灯光/回迁/地库车流）
- **否决**：2% 渗透率下可渗透人口 **< 600**；教练可达性差；房租 **> 30%**（目标 **20–30%**）
- 外立面广告位、商场指引位须提前确认

**招商（已答）** → 同文件 §二；踩坑见 `references/pitfalls/franchise-leverage.md`

- 杠杆：≤50 万店 ≤1 倍，>50 万 ≤0.5 倍；教练加盟谨慎
- 须操盘或 8k–12k 热爱健身的店长；长周期心智；签约前核对分账/对账/打款
- 反例：教练凑钱借贷新开店亏损；对照：低价接盘+亲自上课可跑通

### P6. 教练激励

**必须绑定：**

- 体验课转化
- 消课
- 满意率
- 复购率

**禁止绑定：**

- 与教练用户数**不相关**的客单转化（避免扭曲拉客/压价行为）

### P7. 反常识 TOP 3

1. **渗透率与自助馆增长（2025–2026）**  
   用户渗透率在增加；自助健身房数量约增 40% 的情况下，店均用户数仍普遍创新高，私教转化率未明显下降；私教客单下降 → **以价换量**时代。  
   *边界*：需结合城市线、品牌定位核对本地数据。

2. **课程复杂度 vs 小白接受度**  
   多数钻研课程的人**高估**小白用户接受程度。  
   *边界*：进阶用户、专项工作室可另论。

3. **健身房基础设施化 + 差异化像咖啡**  
   健身人口加速后，「健身房」趋向**水电煤**式基础设施；差异化会像**咖啡店**一样涌现；卷价格、地段、器械都会发生，**不同生态位与价格带的竞争才刚开始**。  
   *边界*：一线 vs 下沉、综合馆 vs 垂直馆节奏不同。
