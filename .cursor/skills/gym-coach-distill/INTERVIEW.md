**目的**：把你的行业判断、方法论、踩坑与决策树写成可检索文本，供 Agent 蒸馏进 `SKILL.md` 与 `references/`。

**填写方式**：复制文末 **回答模板** 为同目录 `ANSWERS.md`；建议顺序：**PRIORITY.md → 模块 G/I → A–F → H**。

**书写提示**：优先写「数字 + 阶段 + 动作 + 结果」；可脱敏公司名；框架请自拟命名便于记忆。

## 模块 A：健身连锁（总部 / 规模化 / 品牌标准）

1. 你经历过的连锁规模阶段（店数区间、直营/加盟比）各是什么？**每个阶段的头号矛盾**是什么？
2. 年度/季度 **开店节奏** 如何定？请写出 if-then：什么信号加速开店、什么信号必须踩刹车。
3. 「标准化复制」里，**必须先统一** 的 5 件事（SOP / 视觉 / 定价 / 系统 / 培训等）及统一顺序。
4. **反常识**：行业常说的「先规模后利润」在你经验里何时成立、何时是自杀？边界条件？
5. 多品牌 / 多店型并存时，**组织怎么切**（总部-区域-门店）才不叠床架屋？画一句汇报线即可。
6. **踩坑清单**：总部在规模化过程中最容易犯的 5 个错（各 1 句后果）。
7. 城市线（一线/新一线/下沉）策略差异：**同一指标** 在不同城市如何改阈值？举 2 个指标。
8. 与竞品对标时，**哪些维度值得对、哪些不值得对**？为什么？
9. 关店 / 收缩决策树：触发条件（指标组合）→ 总部动作 → 门店沟通顺序。
10. 品牌危机（舆情、安全事故、大规模客诉）**72 小时** 标准动作清单。
11. **适用边界**：你的连锁判断对「工作室/单店加盟/纯线上」哪些直接失效？
12. 希望 Agent 在「连锁战略」类问题里 **默认先问的 3 个澄清问题**。

## 模块 B：门店运营（单店经营 / 坪效 / 客流 / 生命周期）

1. 门店生命周期划分（如筹备/爬坡/成熟/下滑）及 **每阶段 Top3 运营动作**。
2. **指标与阈值**：判断门店健康/亚健康/危险的 5 个核心指标（定义、口径、绿黄红、周/月频）。
3. 新店爬坡 **90 天日历**：按周列必做事项（招人、预售、体验课、会员结构等）。
4. 客流下滑时 **排查顺序**（if-then 树）：先排除营销 → 再排除价格 → 再排除 …，写到可执行。
5. 坪效 / 面积 / 器械配置：**什么店型** 下多少坪效算合格？不达标先动哪 3 个杠杆？
6. **反常识**：被夸的「搞活动拉新」「降价促销」在什么门店状态下有害？
7. **踩坑清单**：新人店长 / 区域经理最容易犯的 5 个错（附你见过的简短案例）。
8. 会员结构（月付/年付/私教占比）**恶化** 的早期信号与干预动作。
9. 周边竞品新开 / 价格战：**决策树**（跟进 / 差异化 / 收缩）各需要什么数据？
10. 关店 vs 翻新 vs 换址：**三选一** 判断标准各 3 条。
11. 总部巡检 / 神秘客：**有效项 vs 走过场项** 各举 3 例。
12. 用户问「这家店要不要救」时，skill **必问的 5 个问题**。
13. 门店与教练、前台、保洁的 **编制边界**（人店比、高峰排班原则）。
14. **适用边界**：社区店 / 商场店 / 写字楼店 哪些 playbook 不能混用？

## 模块 C：教练运营（私教 / 团课 / 人效 / 编制）

1. 教练团队 **编制模型**（全职/兼职/外包）及「人店比 / 教练数」推荐区间。

3. 教练 **梯队与晋升**（星级/职级）设计原则；晋升与业绩脱钩的部分是什么？
4. **激励方案**：该绑的指标 5 个、不该绑的 5 个（各写原因与翻车案例一句）。
5. 「高业绩但伤品牌」教练的 **识别信号** 与处理流程（沟通→整改→淘汰）。
6. **反常识**：「明星教练带店」「低底薪高提成」「纯课时费」的真实看法与边界。
7. **踩坑清单**：教练运营最常见的 5 个误区（如只盯课时、忽视体验课转化等）。
8. 体验课 → 正价转化：**标准漏斗** 各节点目标转化率（可区间）及低于目标时的动作。
9. 教练流失率高：**if-then** 诊断（薪酬 vs 管理 vs 客源 vs 品牌）各需要什么证据？
10. 团课满班率 / 取消率 / 迟到率：**阈值** 与对排课表的调整规则。
11. 教练培训 **最小必修包**（产品知识 / 销售 / 服务 / 安全）及考核方式。
12. 跨店支援、借调教练的 **利益分配** 规则（避免扯皮）。
13. 用户只问「教练团队怎么建」时，skill **先问的 3 个澄清问题**。
14. **适用边界**：纯私教馆 vs 团课馆 vs 综合馆，哪些 C 模块答案不能通用？
