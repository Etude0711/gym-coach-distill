# 门店财务指标 — 损益表（综合健身房为主）

> 来源：用户访谈 P4-1。团课工作室见文末边界。

## 一、收入

### 1. 会员卡（会籍）

**价格带（按月卡月费）**

| 档位 | 月费区间 |
|------|----------|
| 低 | 99–159 元 |
| 中低 | 160–239 元 |
| 中高 | 240–300 元 |
| 高 | 300 元以上 |

**健康逻辑（参照 Anytime Fitness 等万店级模型）**

- 会籍的**月度实消收入**应能覆盖**固定成本 + 固定人员开支**。
- **在籍会员数 < 450** → 门店**很危险**（红灯级预警，需结合店型/面积核对）。

### 2. 私教

| 指标 | 参考值 | 说明 |
|------|--------|------|
| 转化率 | **10–12%** | 分母 = **新办会员卡用户数**；观察窗口 = 办卡后滚动 **30 天**（例：月新客 100 → 约 10–12 人转化私教） |
| 单节价（课包） | **200–260 元/节** 常见，可更低 | 约 10 年前常见 500–600 元/节 |
| 教练:门店分成 | **约 4:6** | 课包按节 |
| 包月私教 | **2500–3600 元/月** | 教练按节拿固定课时费；超额上课：超出部分折半或象征性 **10 元/节** |
| 包月用户月上课 | **约 12–15 节** | — |

### 3. 团课工作室（边界）

截至 **2026-06** 判断：**规模化不好**；依赖明星教练；用户对品牌忠诚度低。与综合馆模型分开看，勿直接套用下方「理想店」结构。

## 二、理想状态 — 综合健身房成本与利润（示例模型）

**场景假设**：约 **400㎡**；房租约 **3 元/㎡/月**（不含物业费则约 **3.6 万/月** 固定房租）。

### 月度损益骨架（理想态）

| 项目 | 金额/说明 |
|------|-----------|
| **月收入目标** | **15 万** |
| — 会员卡 | 7.5 万 |
| — 私教 | 7.5 万（5 教练 × 30 天 × 5 节/天 × 200 元 × 门店 50% 分成） |
| **月成本** | **8 万**（不含税、不含额外大促营销） |
| — 房租+物业费 | 4 万 |
| — 水电 | 0.8 万 |
| — 店长+社保 | 1.6 万 |
| — 折旧+器械摊销 | 1.6 万 |
| **月净利润** | **约 7 万** |

### 其他成本参考

| 项目 | 参考 |
|------|------|
| 店长薪酬（一二线） | 7000–12000 元/月；大促有提成；波峰波谷拉平约 **1 万** |
| 营销（连锁单店） | 约 **500 元/月/店**；大连锁另有渠道/品牌大投 |
| 装修摊销 | 500–600 元/㎡，**30 个月**折旧 |
| 器械 | 约 **20–30 万**，**30 个月**折旧 |

### 建店与投资

| 项目 | 参考 |
|------|------|
| 加盟建店 | **60–150 万**，可达 **200 万** |
| 回本周期 | 多数 **24–30 个月** |

## 三、资产负债表（先看什么）

| 关注点 | 判断逻辑 | 绿 / 红（摘要） |
|--------|----------|----------------|
| **负债（预收/会员卡负债）** | 最关心 **每月能「消耗」确认的收入金额**（实消） | **绿**：月度实消 ≥ 覆盖门店 **固定成本**；**红**：实消长期盖不住固定成本（与在籍 <450 等联动） |
| **应付款** | 健身房结构相对简单；重点是 **能耗管理** | 异常水电/能耗飙升 → 黄/红，查设备与用电习惯 |
| **固定资产 / 器械折旧** | 市面常 **高估** 器械耐用年限 | 随用户数爬坡，**24–30 个月** 折旧期较合理（与建店回本周期同量级） |

> 装修摊销见上文「30 个月」；器械 **20–30 万 / 30 个月** 与「24–30 月折旧」可一并核对，避免账面利润虚高。

## 四、现金流、合规与季节节奏

### 监管与预收合规（防跑路语境）

| 地区 | 现状（截至用户访谈 2026-06） |
|------|------------------------------|
| **上海** | 对 **单笔金额、有效周期年限、课包节数** 等有管理要求；执法归 **文旅执法大队**；有一定规模企业常先 **约谈** |
| **北京** | 暂无全市严格规范；**个别区域试点** |
| **二线** | 大多 **暂无明确规定** |

设计课包/会籍周期、收款节奏时，**先查当地监管**，再谈销售模型。

### 连锁经营节奏（现金流视角）

- **旺季**：每年 **3–8 月**。
- **平峰 / 淡季**：现金流压力更大，需提前筹划。
- **季度大促**：缓解销售与现金流；周期稳定，用户形成 **「心理账户」**，教练与店长更好动员。
- 大连锁：**平峰期、淡季现金流** 是总部与区域要 **着重看** 的线，不能只看旺季损益。

### 月度经营现金流 — 实操盯盘（淡季/平峰同样适用）

把「钱能不能转」拆成两条产品线看：**会籍**（稳定高毛利）+ **私教消课**（弹性收入）。

#### 1. 会籍（最稳定、毛利率最高）

| 指标 | 绿 / 正常 | 红 / 预警 |
|------|-----------|-----------|
| **在期会员数** | **≥ 450**（结合 400㎡ 等综合馆模型） | **< 450** → 门店很危险 |
| **月拉新** | 日均 **2–3 人** 办卡算正常 | 持续低于该水平需查渠道/转化 |
| **流失（分层）** | 次月流失 **~50%** 正常；第 3 月 **~30%**；第 4 月起趋稳 | 某层断崖偏离历史基线 |
| **留存 / 召回** | 按淡旺季分别看召回策略与效果 | 淡季只靠新客、无召回 → 现金流吃紧 |

> 会籍是现金流「底盘」：在期人数 + 拉新/流失/留存/召回，比单看当月收款更能预判未来 3–6 个月实消。

#### 2. 私教消课（单独拆账）

| 指标 | 绿 / 正常 | 红 / 预警 |
|------|-----------|-----------|
| **教练人均月消课** | **多多益善**；行业实操线：**≥ 100 节/人/月** | **< 100 节/人/月** → 优先怀疑 **私教管理** 问题 |
| **异常诊断** | 看 **哪个教练** 消课异常、**哪些用户** 活跃异常 | 区分：**服务质量下降** vs **客户过多、教练服务不过来** |

#### 3. 新店爬坡 vs 成熟店（现金流预期）

| 阶段/店型 | 参考 |
|-----------|------|
| **新店预售** | 普遍 **20–30 万**；位置好 **40–50 万**；利于 **投资者/加盟商信心** |
| **成熟·商圈店** | 人流可持续，甚至 **5 年+** 仍有自然客流 |
| **成熟·社区店** | 渗透有天花板，约 **2–3 年** 可触达周边核心人群 |

> 社区店勿用商圈店的「第五年还在涨」外推；预售强弱也影响爬坡期现金跑道。
