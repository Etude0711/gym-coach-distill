# 踩坑 — 加盟与杠杆

## 教练凑钱高杠杆新开店

- **场景**：多名教练存款 + 借贷合开新店。  
- **后果**：亏损叠加个人债务与负面情绪。  
- **Skill 建议**：教练加盟单独提高风控；杠杆见 `../judgments/franchise.md`（≤50 万 ≤1 倍，>50 万 ≤0.5 倍）。

## 教练接盘 vs 教练新开店

| | 新开店（凑钱） | 低价接盘 + 亲自上课 |
|--|----------------|---------------------|
| 风险 | 高（选址、爬坡、杠杆） | 中（取决于接盘价与存量会员） |
| 关键能力 | 操盘、店长、现金跑道 | 本人消课、课时费 + 分红 |
| 勿混淆 | 会上课 ≠ 会开店 | 成功不可复制到新店模型 |
