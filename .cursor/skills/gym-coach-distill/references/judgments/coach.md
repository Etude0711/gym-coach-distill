# 教练运营 — 激励绑/不绑

## 必须绑定的指标

- 体验课转化
- 消课
- 满意率
- 复购率

**原因（摘要）**：对齐「获客 → 交付 → 留存」全链路，避免只冲短期收入。

## 禁止绑定的指标

- 与**教练用户数不相关**的客单转化

**原因（摘要）**：易诱发与用户数无关的压价、抢单或扭曲拉客，损伤长期人效与品牌。

## 人均消课红线

- **< 100 节 / 教练 / 月** → 优先排查私教管理（非先怪市场）。
- 消课「多多益善」；异常时拆到 **教练维度** 与 **用户活跃维度**，区分服务质量下降 vs 过载。

## 澄清后再答（占位）

用户只说「教练流失高」时，先问：店型/编制？流失集中在入职多久？激励方案最近是否改过？（完整清单见 `playbooks/clarify-questions.md`）
