# 门店经营 — 三表诊断法

判断门店经营情况时，按**财务报表逻辑**依次看：

| 表 | 核心问题 |
|----|----------|
| **资产负债表** | 结构是否健康（负债、预付、固定资产等） |
| **损益表** | 是否赚钱、毛利与费用结构是否合理 |
| **现金流量表** | 经营能否持续、是否靠输血存活 |

**损益表**已写入 [`../metrics/store-financials.md`](../metrics/store-financials.md)。关键红灯：**在籍会员 < 450**（需结合面积/店型）；私教转化 **10–12%**（新办卡用户、滚动 30 天）。

**资产负债表** + **现金流/合规/季节** → 同文件 §三、§四；季节节奏见 [`../playbooks/seasonality.md`](../playbooks/seasonality.md)。

**现金流盯盘**：在期会员、拉新/流失分层、私教人均消课（<100 节/月）、预售与店型 → `../metrics/store-financials.md` §四。

## 与「单点 KPI」的区别

不要只用签到、客流等过程指标替代三表；过程指标用于解释三表异常，不替代三表结论。
