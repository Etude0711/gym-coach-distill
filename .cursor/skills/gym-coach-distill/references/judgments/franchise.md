## 一、选址

| 类型 | 说明 |
|------|------|
| 社区、商业综合体、商场、园区、写字楼 | 均属可选物业形态 |
| **优先** | **商场**、**商业综合体** |
| 租金趋势（2025 年后） | 普遍下降 **40–50%**（谈判窗口） |

## 实地调研清单（贝壳 + 夜访）

1. **贝壳**：周边小区 **密度**、**房价** → 估 **入住率**、**人口基数**  
2. **夜访**：看 **晚上灯光**（真实入住）  
3. **回迁房** 识别：影响消费结构与稳定性  
4. **地下室车辆数量** → 辅助判断住户规模与消费力  

## 一票否决

| # | 条件 |
|---|------|
| 1 | **预估可渗透人口 × 2% 渗透率 < 600 人** → **不开店**（即有效健身人口池过小） |
| 2 | **教练可达性差** → **不开店**（教练是交付核心，通勤/驻店成本过高则模型不成立） |
| 3 | **房租 / 预估收入 > 30%** 或长期压线 → 否决或重谈（**控制在 20–30%**） |

## 签约前必须确认

- **外立面广告位**  
- **商场内指引/导视位置**  

> 未提前书面确认 → 开业后获客成本不可控，视为选址流程未完成。

## 二、招商 — 加盟商筛选

### 加分项（摘要）

| 维度 | 加分 |
|------|------|
| 投资心智 | 赚 **成长期** 也赚 **长期** 的钱；心理预期稳定 |
| 过往经验 | 有 **投资经验** 加分 |
| 扩张模式 | **多店 + 纯财务投资不管店** 可以是理想模式，但前提是：有 **稳定、听从平台** 的门店经营者 / 店长 |

### 一票否决 / 强约束

| # | 维度 | 标准 |
|---|------|------|
| 1 | **资金结构 / 杠杆** | 看 **现金流能力**；加盟费与店型不同，比例要求不同：<br>• 建店 **≤50 万**：杠杆 **≤1 倍**<br>• 建店 **>50 万**：杠杆 **≤0.5 倍**<br>• **教练自有资金加盟** → **谨慎**（见反例） |
| 2 | **操盘与店长** | 须能 **亲自操盘** 或配备 **靠谱店长**；店长宜偏年轻，薪资约 **8k–12k/月**，**热爱健身** |
| 3 | **周期心智** | 不接受健身 **长周期**、只想快钱 → 不匹配 |
| 4 | **总部标准** | 装修、人效、数据上报、**流程与费用占比**、**分账 / 对账 / 打款** → **签约前书面核对**，不接受则否 |
| 5 | **多店 / 甩手掌柜** | 可以多店、可以投资人不驻店，但 **必须有** 稳定执行层；不能「只有钱、没人听平台」 |

### 案例（脱敏）

**反例 — 教练凑钱加杠杆开店**

- **背景**：3–4 名教练 **存款 + 借贷** 合开门店。  
- **结果**：门店亏损，对教练个人 **财务与情绪冲击大**。  
- **教训**：教练加盟 **杠杆与现金流** 要更严；缺操盘与缓冲垫时，勿用「自己会上课」替代选址与经营模型。

**对照 — 教练接盘亏损店**

- **背景**：教练 **低价接盘** 业绩下滑门店，**亲自上课**。  
- **结果**：课时费收入 + 门店分红，可跑通。  
- **边界**：这是 **接盘 + 亲自交付** 模型，不是「凑钱新开店」模型；低价、存量会员、本人消课能力是关键变量。
