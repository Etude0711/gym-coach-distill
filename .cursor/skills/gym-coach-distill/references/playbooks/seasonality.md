# 季节与大促节奏（现金流）

## 旺季

- **3–8 月** 为行业旺季（综合健身房连锁语境）。

## 淡季 / 平峰

- 现金流与销售压力显著上升；连锁总部与区域应 **提前** 做现金与人力筹划，勿仅用旺季损益外推全年。

## 季度大促

- 周期稳定 → 用户形成 **心理账户**（知道什么时候买划算）。
- 缓解 **销售漏斗 + 现金流**；便于动员教练、店长。

## 与财务三表的关系

- 淡季看 **现金流量表** 与 **预收负债的月度实消**，不能只看损益表当月利润。
- 上海等地 **预收/课包合规** 影响收款节奏，见 `../metrics/store-financials.md` §四。
